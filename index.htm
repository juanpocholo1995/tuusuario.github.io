Ejemplo de código

A continuación un pequeño script en JavaScript que actualiza el reloj y un bloque de ejemplo en Python formateado en un <pre>.

JavaScript (interacción)

// Actualiza reloj cada segundo
function updateClock(){
  const el = document.getElementById('clock');
  const now = new Date();
  el.textContent = now.toLocaleTimeString();
}
setInterval(updateClock,1000);
updateClock();

Python (ejemplo)

# Guarda esto en un archivo .py y ejecútalo con Python 3
x = 42
mensaje = f"El valor de x es {x}"
print(mensaje)
